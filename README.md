
# ShopEase

A simple e-commerce frontend built with React.

## About

ShopEase is a multi-page shopping app I built to practice React concepts like
component structure, React Router, and the Context API. It has a product listing
page with search and filters, individual product pages, and a working cart.

There's no backend or payment system — it's purely a frontend project with local
product data.

## Features

- Browse 12 products across different categories
- Search products by name
- Filter by category
- View product details
- Add to cart, increase/decrease quantity, remove items
- Cart total updates automatically
- Responsive layout for mobile and desktop

## Tech Used

- React (with Vite)
- React Router v6
- Context API for cart state
- Plain CSS

## How to Run Locally

```bash
git clone https://github.com/yourusername/shopease.git
cd shopease
npm install
npm run dev
```

Then open http://localhost:5173 in your browser.
